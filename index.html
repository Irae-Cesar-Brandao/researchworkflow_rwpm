RWPM V1 – Quick User Guide

Research Workflow & Publication Manager (RWPM) is an experimental academic tool designed to assist researchers in tracking manuscripts, submissions, editorial communications, similarity analyses, AI detection reports, and publication workflows.

Basic Commands

Action Description
Add Project Create and save a new project.
Edit Modify existing project information.
Delete Remove a project permanently.
Export JSON Create a backup copy of all projects.
Import JSON Restore previously exported project data.
Theme Switch between light and dark mode.

Project Fields

Field Purpose
Control Number Internal project identification code.
Title Project or manuscript title.
Authors Authors and contributors.
Status Current publication stage.
Similarity Analysis Similarity and originality assessment records.
AI Detection AI-generated content assessment records.
Word Count Current and target manuscript length.
Pending Items Tasks, revisions, and editorial requests.
Email Log Editorial correspondence history.
Timeline Chronological project record.

Data Backup Recommendation

All project information is stored locally in your browser using Local Storage technology. Users are strongly advised to export and save JSON backup files regularly, especially before browser updates, device changes, system maintenance, or data-cleaning operations.

Failure to maintain backup copies may result in irreversible loss of project information.
