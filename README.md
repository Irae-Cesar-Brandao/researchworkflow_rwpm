<h2>RWPM V1 – Quick User Guide</h2>

Research Workflow & Publication Manager (RWPM) is an experimental academic tool designed to assist researchers in tracking manuscripts, submissions, editorial communications, similarity analyses, AI detection reports, and publication workflows.

<h3>Data Backup Recommendation

All project information is stored locally in your browser using Local Storage technology. Users are strongly advised to export and save JSON backup files regularly, especially before browser updates, device changes, system maintenance, or data-cleaning operations.


 Failure to maintain backup copies may result in irreversible loss of project information.
